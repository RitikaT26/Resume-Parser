# Resume Parser AI + Job Match Scoring System

An AI-powered Resume Parser project built using Python, NLP, and Machine Learning.

The system extracts text from resumes, identifies skills, predicts resume category, and compares resumes with job descriptions using TF-IDF and Cosine Similarity.

---

# Features

- Upload PDF/DOCX Resume
- Resume Text Extraction
- Resume Category Prediction
- Skill Extraction
- Resume Match Score
- Missing Keyword Detection
- Streamlit User Interface

---

# Technologies Used

- Python
- Streamlit
- NLTK
- scikit-learn
- TF-IDF Vectorization
- Naive Bayes
- Cosine Similarity

---

# Dataset Used

Kaggle Resume Dataset

Dataset contains:
- Resume categories
- Resume text information

Example Categories:
- Data Science
- Python Developer
- HR
- Java Developer
- DevOps Engineer
- Web Designing
